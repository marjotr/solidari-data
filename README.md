O **solidari-data** é um projeto desenvolvido em Python que nasceu no meu coração com o desejo de colocar a **Ciência de Dados** a serviço do próximo. Ele atua como um **Monitor de Doações e Impacto Social**, idealizado para ajudar igrejas, missões e pequenos projetos sociais a organizarem suas ações de solidariedade com excelência, zelo e amor cristão. 🤝💖

---

## ⛪ Contexto do Ministério e Aplicação Real

Este projeto foi especialmente pensado para apoiar e otimizar ações como o **Quilo do Amor**, um projeto social da **Igreja Batista Atitude** (da qual sou membra). 

Nesta campanha de arrecadação de alimentos, os fiéis são convidados de forma voluntária a doar 1 kg de alimento não perecível (ou mais) durante os cultos de celebração de Santa Ceia. Todas as doações são recolhidas com muito carinho pelo **Instituto Assistencial Atitude** e destinadas a abençoar famílias em situação de vulnerabilidade social.

O *solidari-data* serve como uma ferramenta de apoio tecnológico para monitorar a entrada desses alimentos e calcular o impacto gerado a cada mesa assistida.

---

## 🚀 Funcionalidades para Servir com Excelência

* 🌸 **Zelo nos Registros:** Organiza e estrutura as doações recebidas (como alimentos não perecíveis do Quilo do Amor, bíblias e itens de higiene) transformando-as em dados claros.
* 📊 **Multiplicação e Impacto:** Consolida relatórios automáticos mostrando o total de quilos ou itens arrecadados e calcula quantas famílias foram alcançadas e abençoadas por cada ato de amor.
* 💾 **Guarda Segura (JSON):** Salva todas as informações em um arquivo local automaticamente, garantindo a boa mordomia e organização dos dados sem perder nenhum registro.

## 📦 Como Instalar e Rodar o Projeto

Se você deseja usar ou testar este monitor na sua comunidade ou igreja, siga estes passos:

### 1. Clonar o repositório
Abra o seu terminal (ou GitHub Desktop) e baixe o projeto:
```bash
git clone https://github.com/marjotr/solidari-data.git
```

### 2. Iniciar o Monitor
Entre na pasta do projeto e execute o comando:
```bash
python monitor.py
```

## 🔒 Mordomia e Proteção de Dados

Alinhada com os princípios bíblicos de integridade e cuidado com o próximo:
* O arquivo `quilo_do_amor_dados.json` está protegido no `.gitignore`. 
* Isso garante que as informações reais das pessoas que doaram ou receberam ajuda permaneçam em total privacidade e **nunca** sejam expostas na internet pública.
